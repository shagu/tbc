Grid Plugins

All-in-One

This package includes every plugin listed below. If you don't have Grid already installed, you can do so by clicking here.

Grid_QuickHealth

Makes grid QuickHealth aware: in short -- this means that health bars in Grid would change their values more often if your ping is good. For more information, visit this page.

GridAlert

Alerts for Grid. When a player in your group/raid gets a debuff that you can get rid of, you hear a sound.

GridAutoFrameSize

Automatically resizes frame layout depending on group/raid size.

GridBuffIcons-1.0

This module adds small buff (debuff) icons in the Grid unit frames, just like the Blizzard RaidUI pullout buttons.

GridBuffIcons2-1.0

Same as above, I think. I haven't personally tried either this or the one above, so testing might be required to see what the difference between the two is.

GridCooldownText

Displays the remaining time on grid icon's cooldown animation.

GridDynamicLayout

GridDynamicLayout is an additional layout for Grid that attempts to dynamically sort your raid.

GridDynamicZoneSwitch

Configure which layout or profile to load on entering a zone.

GridIndicatorSideIcons

Adds four icon indicators to Grid (top, bottom, left and right).

GridIndicatorText

Allows you to add text instead of icons/dots on the frames. (e.g. Stacks of your Earth Shield as a number.)

GridLayoutForHealers

Adds common healer layouts for Grid.

GridLayoutPlus

This plugin is made to be a set and forget layout manager. You just set your layout parameters and the layout will automatically change depending on what units are available. The grid width (or height if you changed your layout direction) will auto-expand depending on the raid size, or max subgroup used. If you select you want to see pets, column(s) will be added depending on if you have hunters or warlocks in your raid. Tanks will be shown if you enable the option AND Blizz/oRA/CTRA tanks are set. You can change the sort order of the classes, pets & tanks. You can also change the number of units in each column. For example, you could set it to 20 and in AV you will see 2 rows of 20.

GridManaBars

Allows you to see resources (mana, energy, rage) on player frames.

GridoRA2Layouts

Adds oRA2 MT and pets Layouts to Grid.

GridSideIndicators

Plugin to Grid that adds 4 indicators, one to each of the sides of Grid frames: top, bottom, left and right.

GridStatusAFK

Adds AFK status to Grid frames.

GridStatusControlPipes

This addon provides further control possibilities for any other Grid status. For further information, visit here.

GridStatusGroupnumber

Group number text for Grid.

GridStatusHealingReduced

GridStatusHealingReduced adds two new statuses to Grid: one for debuffs which reduce healing received, and one for debuffs which effectively prevent healing received. These statuses are configurable in all the usual ways, and can be assigned to any indicator.

GridStatusHealPriority

Status indicator that shows you your healing priorities.

GridStatusHealthPercent

Displays health as a precent.

GridStatusHostileUnit

Status indicator that shows if a unit is hostile. (MC'ed players etc.)

GridStatusHots

Separate indicator with timer (and counter) for: Rejuvenation, Regrowth, Lifebloom, Earth Shield, Renew, Prayer of Mending, Power Word: Shield, Weakened Soul, Inspiration, and Ancestral Fortitude.

GridStatusIncomingHeals

Status indicator that shows amount of incoming heals to the target, using IncomingHealsLib.

GridStatusLifebloom

Custom status indicator for grid to show the time remaining on your own lifebloom.

GridStatusLineOfSight

Adds "line of sight" status to Grid (after the error occurs). The status is disabled seconds later. The idea being if someone you're healing is LOS, you can dim their frame's transparency temporarily or indicate that you need to maneuver yourself into line of sight.

GridStatusMissingBuffs

Shows missing buffs and handles welfare and deluxe buffs as one (motw/gotw, blessings, etc.). This lets you see when you need to apply a buff.

GridStatusMTs

Adds Blizzard, oRA2, oRA3, CT_RA MainTank or Dungeon Finder Tank icon (shield or MT, 1, 2 ... 10) or text (MT or MT1, MT2 ... MT10) status to Grid.

GridStatusParty

Shows group leadership roles in Grid.

GridStatusRaidDebuff

Adds a status to Grid for debuffs applied by Raid Bosses.

GridStatusRaidIcons

Adds Raid Target Icon status to Grid (Player or Player Target).

GridStatusReadyCheck

Allows you to see the results of a ready check in Grid.

GridStatusRes

GridStatusRes displays resurrections and soulstones in Grid.

GridStatusShield

Provides the remaining amount of your shields on a unit.

GridUnitMenu

This addon is a utility extension/plugin for Grid. The only thing it does is adding a right-click unit menu to your grid raid frames.
